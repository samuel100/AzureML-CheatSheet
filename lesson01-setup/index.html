    Rough notes

    • Set up your Azure ML Workspace
    • Download the python sdk with e.g.

    Simply run: 

    pip install --upgrade azureml-sdk
    Go here for more details on installing the AzureML Python SDK.

    Create an env.yml file 

    name: aml-blitz

channels:
    - defaults
    - conda-forge

dependencies:
    - python=3.6
    - pip
    - pip:
        - azureml-sdk
    and run 
    conda env create -f env.yml
    Go here for more details on installing the AzureML Python SDK.
